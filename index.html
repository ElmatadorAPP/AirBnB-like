Wink

Wink is a Web application that allows individuals to exchange their homes (apartment, house, loft, etc.) for a short vacation (a day to a few weeks). These exchanges are free, but the counterpart is that each owner defines some servies that the guests must provide and some constraints/limits that they must respect.
